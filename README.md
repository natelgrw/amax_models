# AMAX-1: UV-Vis 位<sub>max</sub> Prediction for LC-MS Applications

**AMAX-1** is an open source dataset designed to assist machine learning models in small molecule **UV-Vis absorption maxima (位<sub>max</sub>)** prediction and LC-MS **compound characterization workflows**.

##  The AMAX-1 Dataset

The **AMAX-1 dataset** contains:

- **40,026 moleculesolvent combinations**, the largest 位<sub>max</sub> dataset of its kind to date
- Experimentally measured **位<sub>max</sub> values** curated from public datasets, benchmark papers, and literature

The dataset is actively expanding with new experimental 位<sub>max</sub> values from the **Coley Research Group at MIT**, ensuring AMAX-1 remains a growing resource for optical property prediction.

The dataset is designed for use in:

- Estimating 位<sub>max</sub> for new compoundsolvent pairs
- Aiding in peak assignment during LC-MS method development
- Supporting generalized chromophore engineering and spectral ML workflows
